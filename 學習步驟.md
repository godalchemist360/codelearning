# 全端開發學習計劃

## 專案概述：「個人書籤筆記」應用程式

我們會一起建立一個簡單的書籤筆記網站，功能包含：
- 新增、查看、刪除書籤（網址 + 標題 + 備註）
- 透過網頁介面操作
- 資料儲存在 PostgreSQL 資料庫

這個專案小而完整，能讓你體驗完整的「前端 → 後端 → 資料庫」流程。

---

## 技術架構圖

```
[使用者瀏覽器]
      |
      | HTTP 請求
      v
[Next.js 前端]  ←→  [C# .NET API 後端]  ←→  [PostgreSQL 資料庫]
  (顯示畫面)         (處理邏輯)               (儲存資料)
      
      ↑
      整體部署到 Azure
```
